## 🐛 Issues & Data Corrections

Found an error in the data or a bug in the dashboard?

1. [Search existing issues](https://github.com/juanmoisesd/mental-health-specialists-iberoamerica/issues) to avoid duplicates
2. [Open a new issue](https://github.com/juanmoisesd/mental-health-specialists-iberoamerica/issues/new/choose) using the appropriate template

Please include:
- A clear description of the problem
- The specific data point or feature affected
- Any relevant sources or references
